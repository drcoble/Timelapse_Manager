Network access (camera reachability)

Cameras live on private networks, so for safety the server refuses to contact a private address unless that network has been explicitly allowed here. If adding a camera fails with "address is not within any admin-allowed subnet", add its network below (for example 10.1.16.0/24). Loopback, link-local, and cloud-metadata addresses are always blocked and cannot be allowed. Changes take effect immediately — no restart needed.
